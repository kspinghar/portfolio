Task Scheduler App

Python desktop app for managing and prioritizing daily tasks.

OVERVIEW

This project is a lightweight desktop application for task management. It allows users to add, view, and prioritize tasks using a clean and functional GUI built with Tkinter.

KEY FEATURES & IMPLEMENTATION

Create, delete, and edit tasks with deadlines
Prioritize tasks based on importance and due date
Simple drag-and-drop interface for reordering
Local storage using plain text or JSON

TECHNOLOGIES USED

Python, Tkinter, JSON, Object-Oriented Programming

CHALLENGES

Designing an intuitive GUI with minimal dependencies
Keeping the interface responsive and user-friendly
Balancing features with simplicity for daily usability

LEARNINGS & IMPACT

This project helped me improve my GUI design skills and reinforced core OOP principles in Python. It taught me how to structure small desktop tools for real-world utility.
